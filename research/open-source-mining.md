# Open-source projects worth mining

## Eww

Closest Linux-native conceptual/codebase target.

Good ideas:

- Rust + GTK app structure
- Window/widget lifecycle
- CSS styling
- Config reload
- User-facing widget config model
- Examples/gallery culture

License: MIT. Good source for patterns and possibly small reusable pieces with attribution.

## Waybar

Good ideas:

- Linux status modules
- Battery, CPU, memory, network, audio, MPRIS, Bluetooth, tray
- CSS styling
- Custom command modules
- Multi-output behavior

License: MIT. Good source for module design.

## Conky

Good ideas:

- System metric collection
- Graphs and progress bars
- Low-resource refresh loop
- Lua/Cairo extension ideas

License caution: treat as idea-first unless matching license obligations.

## Übersicht

Good ideas:

- Widget folder model
- `command` + `refreshFrequency`
- HTML/JS widget rendering
- Live reload
- Public widget gallery warning about arbitrary code

License: MPL-2.0-or-later. Good conceptual cousin.

## Rainmeter

Good ideas:

- Measure/meter split
- Skin package format
- Theme ecosystem
- Plugin model

License: GPL-2. Be careful with code reuse.

## AGS / Astal

Good ideas:

- Desktop shell services
- Notification/audio/network/service feeds
- Scriptable shell architecture

License: MPL-2.0. Idea mining is easy; direct code reuse depends on NeoDash license.
